# Fieldwatch

Fieldwatch is a clean, minimal sports news tracker built with React, Vite, and plain CSS.

## Features

- Feed page with recent sports articles sorted newest first
- Followed team chips for Arsenal, Golden State Warriors, and Man United
- Category filter pills for transfers, injuries, match reports, and press
- Schedule page with a seven-day strip and match cards
- Players page with followed and suggested player cards

## Run locally

```bash
npm install
npm run dev
```

To create a production build:

```bash
npm run build
```
